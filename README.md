# 🚀 Intelligent Object Detection System

A modular Computer Vision application built using **Python**, **OpenCV**, **YOLOv8**, and **ByteTrack** for intelligent object detection, real-time tracking, analytics, and reporting.

---

## 📖 Overview

This project demonstrates a complete real-time object detection pipeline capable of processing:

- Images
- Videos
- Live Camera Streams

The system performs high-performance object detection using YOLOv8 while maintaining object identities through ByteTrack.

It also provides live analytics, FPS monitoring, object counting, logging, and automatic report generation.
